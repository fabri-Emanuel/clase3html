Escribir en varias lineas

Para escribir en varias lineas de codigo. Lo que haremos es sostener la tecla ctrl + alt y luego presionar la tecla flecha hacia arriba o hacia abajo


Escribir HTMIL de manera rapida

Vamos a escribir el nombre de la etiqueta y luego damos un Enter


Comentarios

Para escribir lineas de comentario vamos a ir a Editar hasta la opcion alternar comentario de linea o bien, verificar el acceso por teclado asignado.


Crear un documento HTML basico rapidamente

Para poder crear una estructura HTML rapido, precionamos SHIFT + 1 Y damos un Enter o click sobre la primera opcion que nos ofrece VSC
